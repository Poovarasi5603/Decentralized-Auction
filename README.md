Getting Started

Try running the below commands :

truffle init
truffle compile

create a JavaScript migration script,
01-Auction-deployment.js
02-AuctionCreator-deployment.js

Deploy on truffle blockchain at localhost 9545,
truffle develop
truffle migrate
